# 旅游城市关键词分析系统 python1101

## 项目概述

旅游城市关键词分析系统是一个基于Python语言的Web网站项目，采用Django框架和MySQL数据库，构建在B/S架构之上，遵循MVC设计模式。本项目旨在为用户提供旅游城市的热度分析、热门景点及小吃推荐等功能。

## 技术栈

- **架构：** B/S、MVC
- **系统环境：** Windows
- **开发环境：**
  - PyCharm
  - JDK
  - Python
  - MySQL
- **技术：**
  - Python
  - Django
  - MySQL

## 功能模块

### 前台系统

- **登录与注册界面：**
  - 用户注册：未注册用户可在此完成注册流程。

- **旅游城市关键词分析系统首页：**
  - 热门城市推荐：展示推荐旅游城市。
  - 旅游城市热度分析：提供城市热度排名入口。
  - 热门景点分析：链接至热门景点详情页面。
  - 热门小吃分析：展示小吃街及推荐小吃入口。

- **旅游城市热度分析界面：**
  - 显示城市热度排名：左侧列表展示。
  - 评分统计信息展示：右侧环形图展示。

- **热门景点分析页面：**
  - 详细介绍热门景点：以图文列表形式展示。

- **热门小吃分析界面：**
  - 展示热门小吃街：通过图片及描述。
